8.2 InfoVole – getting extra information about a Vole search

8.2-1 InfoVole
‣ InfoVole( info class )

InfoVole is the primary info class for Vole. See Reference: Info Functions for a description of info classes in GAP.

The default info level is 0. Most info messages are given at level 2, and additional messages are given at level 4.
